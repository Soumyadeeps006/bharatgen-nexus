# BharatGen: Enterprise OS

BharatGen is an enterprise-grade platform engineered to orchestrate large-scale data workflows, custom analytics, and intelligent operations. Inspired by industry-leading enterprise operating systems, it provides a centralized command center for data engineering, predictive modeling, and system monitoring.

## Features

- **Executive Ontology Dashboard**: A real-time executive view of data pipelines, node health, and overall system resource utilization.
- **Modern Aesthetic**: Built with a highly polished, glassmorphic UI overlaying a dark mode environment, prioritizing user experience and data legibility.
- **React & Vite**: Built for performance with React and bundled via Vite for lightning-fast HMR and optimized production builds.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.x or later recommended)
- `npm` or `yarn`

### Installation

1. **Clone the repository** (if hosted remotely) or navigate to the project directory:
   ```bash
   cd bharatgen
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:5173`.

## Architecture Overview

The application is structured for scalability and maintainability:

- `src/components/`: Reusable UI elements such as `Sidebar`, `TopNav`, and `Layout`.
- `src/pages/`: Main application views like the `Dashboard`.
- `src/index.css`: Defines the global design system, CSS variables, and utility classes (e.g., `.glass-panel`).
